- **Term:** Spring 2026

## Learning goals
By the end of the course, students will be able to:
1. Translate a physics model into working code.
2. Visualize and interpret computational results.
3. Use numerical methods (linear algebra, ODE solvers, fitting, Monte Carlo).
4. Communicate results with clear plots, explanations, and reproducible scripts.
